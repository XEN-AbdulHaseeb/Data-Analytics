# Employee Turn Over/Churn Analysis

We have a dataset containing historical employee data and data for a pilot batch

We are tasked to predict which employees will churn/turn over

## Tools Used

Big Query

Google Colab Research Notebook for analysis using python

Looker studio for visualization
